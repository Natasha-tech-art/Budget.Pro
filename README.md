# BudgetPro - Budget Planner Website
## Project Description
   BudgetPro is a web-based budget planner that helps users manage their finances effectively.
   Users can track income,record expenses and monitor their balance in real-time.
   The website also provides learning resources on financial management and different pricing plans.

   ## Features
   1. Home page - Introduction and features
   2. About page - Explains the purpose of the app
   3. How It Works - Step by step guidance
   4. Plans & Pricing - Contains filtering and search functionality
   5. Learn page - Contains financial articles,search search and filtering
   6. Dashboard - Adds income,add expenses and view total balance
   7. Login Page - Contains validation and show/hide password
   8. Contact Page - Form validation
